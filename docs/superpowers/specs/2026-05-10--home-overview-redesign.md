# 总览页改版设计

日期：2026-05-10

## 目标

将总览页从简单占位信息升级为真正的首页入口，采用“混合型”结构：

1. 顶部提供欢迎信息和页面定位
2. 中部提供清晰的快捷入口
3. 底部提供简洁的数据概览和最近内容

## 页面结构

### 1. 欢迎区

- 展示当前用户欢迎语
- 简要说明系统当前核心能力
- 提供主行动按钮，优先引导进入资料页

### 2. 快捷入口区

- 提供四个快捷卡片：
  - 资料
  - 知识库
  - 我的上传
  - 个人中心
- 每个卡片有简短说明和直接跳转

### 3. 简洁概览区

- 展示平台文档总量
- 展示当前用户上传数和积分（登录后）
- 展示当前搜索维度或系统状态

### 4. 最近内容区

- 展示最近更新的少量资料
- 直接链接到详情页
- 作为总览页的轻量内容预览

## 设计原则

- 保持现有视觉语言，不做大范围风格跳变
- 强化首页作为“导航入口”的角色
- 信息层级清晰，避免过度堆积
- 登录和未登录状态都能正常显示
