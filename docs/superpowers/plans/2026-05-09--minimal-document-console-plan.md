# Minimal Document Console Plan

## Scope

Refactor the current campus-sharing app into a minimal document console with:

- only two primary navigation items: `总览` and `资料`
- a document model that stores only `description + file`
- a document table workflow for search, upload, preview, and download
- removal of title, subject, term, resource type, and tag concepts from the document workflow
